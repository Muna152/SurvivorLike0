# 游戏设计文档 (Game Design Document)

## 项目名称：暗夜幸存者 (Night Survivors)

---

## 1. 游戏概述

| 属性 | 说明 |
|------|------|
| **游戏类型** | Roguelite 生存动作游戏 |
| **参考游戏** | Vampire Survivors, Brotato, Holocure |
| **平台** | PC (Steam) / 移动端 |
| **视角** | 俯视角 2D |
| **核心玩法** | 移动躲避 + 自动攻击 + 升级选择 |
| **单局时长** | 15~30 分钟 |
| **目标受众** | 休闲玩家、Roguelite 爱好者、追求爽快感的玩家 |
| **美术风格** | 像素风 / 扁平化卡通 |

### 核心玩法一句话描述

> 玩家操控角色在黑暗中生存，武器自动攻击海量敌人，通过升级获取更强大的武器和道具，活到最后即为胜利。

---

## 2. 核心游戏循环

### 2.1 单局流程

```
角色选择 → 进入关卡 → 战斗生存(30min) → 胜利/死亡 → 结算 → 返回大厅
```

### 2.2 局内循环

```
移动躲避敌人
    ↓
武器自动攻击
    ↓
击杀敌人 → 掉落经验宝石
    ↓
拾取经验 → 累积升级
    ↓
升级时暂停 → 从3个选项中选择（新武器/武器升级/被动道具）
    ↓
角色变强 → 面对更强敌人
    ↓ （循环）
```

### 2.3 局外循环

```
结算获得金币
    ↓
在大厅购买永久属性加成
    ↓
解锁新角色/新武器
    ↓
以更强的起点开始下一局
```

## 3. 角色系统

### 3.1 基础属性

| 属性 | 说明 | 基础值 |
|------|------|--------|
| **HP** | 生命值，归零则死亡 | 100 |
| **移速** | 移动速度 (单位/秒) | 3.0 |
| **护甲** | 减少受到的伤害 | 0 |
| **拾取范围** | 自动拾取掉落物的半径 | 2.5 |
| **幸运值** | 影响稀有掉落和升级选项品质 | 0 |
| **再生** | 每秒恢复HP | 0 |
| **伤害加成** | 全局伤害乘数 | 1.0x |
| **冷却缩减** | 武器冷却时间乘数 | 1.0x |
| **范围加成** | 武器效果范围乘数 | 1.0x |
| **弹数加成** | 投射物数量加成 | 0 |

### 3.2 角色列表（初始设计 5 个）

| 角色 | 起始武器 | 特殊被动 | 特点 |
|------|----------|----------|------|
| **勇者** | 飞剑 | HP+20% | 均衡型，适合新手 |
| **法师** | 能量球 | 冷却-10% | 高攻速，低HP |
| **骑士** | 旋转盾 | 护甲+2 | 高防御，低移速 |
| **游侠** | 飞刀 | 移速+15% | 高机动，多弹数 |
| **牧师** | 圣光 | 再生+0.5/s | 自回复，辅助型 |

### 3.3 等级与成长

- 最高等级：30级（30分钟局每分钟约1级）
- 每级所需经验递增：`所需经验 = 5 + 3 × 等级²`（二次曲线，前期升级快后期慢）
- 升级时游戏暂停，展示3个随机选项
- 属性成长：每级小幅提升HP（+2%）和移速（+0.5%）

## 4. 武器系统

### 4.1 武器类型

| 类型 | 描述 | 代表武器 |
|------|------|----------|
| **投射型** | 向最近敌人发射弹丸 | 飞剑、飞刀、能量球 |
| **轨道型** | 围绕角色旋转的武器 | 旋转盾、光环 |
| **范围型** | 在区域内造成持续伤害 | 圣水、火焰圈 |
| **辅助型** | 不直接伤害但提供增益 | 护盾、磁铁力场 |

### 4.2 武器属性

| 属性 | 说明 |
|------|------|
| **基础伤害** | 每次命中的伤害值 |
| **冷却时间** | 两次攻击的间隔（秒） |
| **效果范围** | 攻击作用半径 |
| **弹数** | 每次攻击发射的投射物数量 |
| **穿透** | 投射物可穿透的敌人数 |
| **弹速** | 投射物飞行速度 |
| **持续时间** | 效果持续时长（范围型/轨道型） |

### 4.3 武器升级

- 每把武器最高 **8级**
- 升级提升：伤害+10%~20%、冷却-5%~10%、弹数+1（特定等级）
- 升级来源：升级选择界面中选取"武器升级"选项

## 6. 敌人系统

### 6.1 敌人类型

| 敌人 | HP | 移速 | 伤害 | 经验值 | 出现时间 | 行为 |
|------|-----|------|------|--------|----------|------|
| **骷髅兵** | 10 | 1.5 | 5 | 1 | 0min | 直线追踪 |
| **蝙蝠** | 5 | 3.0 | 3 | 1 | 0min | 快速冲刺 |
| **僵尸** | 25 | 1.0 | 8 | 2 | 3min | 缓慢但坚韧 |
| **法师** | 15 | 1.2 | 12 | 3 | 5min | 远程投射 |
| **幽灵** | 8 | 2.5 | 6 | 2 | 8min | 穿越障碍 |
| **石像鬼** | 50 | 1.8 | 15 | 5 | 12min | 高HP冲锋 |
| **暗影刺客** | 20 | 3.5 | 10 | 4 | 15min | 隐身+偷袭 |

### 6.2 精英敌人

- 每5分钟出现一波精英
- 外观放大1.5倍 + 特殊颜色
- HP为普通版的5倍，伤害2倍
- 死亡必定掉落大经验宝石

### 6.3 BOSS

| BOSS | 出现时间 | HP | 特殊攻击 |
|------|----------|-----|----------|
| **骷髅王** | 10min | 500 | 召唤骷髅群 + 范围震击 |
| **暗夜领主** | 20min | 2000 | 全屏弹幕 + 传送 + 召唤精英 |
| **死神** | 30min | 5000 | 一击必杀线 + 追踪光束 + 无法被杀死（只能存活到时间结束） |

### 6.4 敌人生成规则

- 生成区域：以玩家为中心，半径 15~25 单位的环形区域
- 生成密度：随时间增长，公式：`基础密度 × (1 + 0.15 × 分钟数)`
- 同屏上限：500个敌人（性能约束）
- 生成间隔：每 0.5~2 秒生成一批（随难度减少间隔）

## 7. 关卡设计

### 7.1 初始关卡：暗夜森林

- 开阔草地为主 + 零散树木和石头障碍
- 地图大小：200×200 单位
- 玩家出生在中心
- 相机跟随玩家

### 7.2 时间节奏

| 时间段 | 难度描述 | 敌人构成 |
|--------|----------|----------|
| 0-5min | 温和 | 骷髅兵、蝙蝠 |
| 5-10min | 上升 | + 僵尸、法师 |
| 10-15min | 困难 | + 幽灵、精英波次、骷髅王BOSS |
| 15-20min | 激烈 | + 石像鬼、更多精英、暗夜领主BOSS |
| 20-25min | 地狱 | 全类型、高密度 |
| 25-30min | 终极 | 死神降临、极限密度 |

### 7.3 掉落物

| 掉落物 | 来源 | 效果 |
|--------|------|------|
| **小经验宝石** | 普通敌人(80%) | +1 经验 |
| **中经验宝石** | 精英敌人/宝箱 | +5 经验 |
| **大经验宝石** | BOSS | +20 经验 |
| **烤鸡** | 敌人(5%几率) | 恢复30HP |
| **宝箱** | 精英/BOSS | 触发武器进化（条件满足时）|
| **金币** | 所有敌人 | 局外货币 |

> **吸附机制**: 经验宝石落地2.5秒后进入真空吸附状态，从8距离内自动飞向玩家，带二次缓入加速效果（起步慢→越来越快）。基础拾取范围2.5。

## 10. 数值平衡原则

1. **前期弱，后期强**：让玩家感受成长
2. **密度递增**：用数量制造压力，不是单个敌人强度
3. **爆发窗口**：进化武器提供高潮体验
4. **30分钟为界**：死神不可击杀，强制结束
5. **运气成分**：幸运值影响升级选项和掉落，保证每次体验不同
6. **数值公式化**：所有数值基于公式，方便全局调整
